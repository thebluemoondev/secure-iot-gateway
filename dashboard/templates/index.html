Luồng 2 — Baseline (không bảo mật)

  1. Bấm nút trạng thái phía trên để tắt mã hoá — server chuyển sang chế độ baseline ngay lập tức, không cần khởi động lại tiến trình.
  2. Từ máy khác, giả vờ không có ai đi qua (che giấu xâm nhập):
    python attacker_sim/baseline_fake_packet.py --host 10.54.93.155
  3. Hoặc giả mạo thẻ hợp lệ để được "cho vào" dù không có ai:
    python attacker_sim/baseline_fake_access.py --host 10.54.93.155 --uid EE134002
  4. Xem kết quả ngay trong "Nhật ký sự kiện bảo mật" bên dưới. Bấm lại nút trạng thái để bật mã hoá lại sau khi demo xong.

Luồng 3 — MITM (can thiệp đường truyền)

  1. Giữ server ở chế độ SECURE (không tắt mã hoá), chạy proxy chèn giữa ESP32 và Server:
    python attacker_sim/mitm_tamper.py --listen-port 9001 --target-host 10.54.93.155 --target-port 9000
  2. Tạm trỏ SERVER_HOST/SERVER_PORT trong ESP32 sang IP máy chạy proxy + cổng 9001, nạp lại firmware.
  3. Kỳ vọng: Server thật trả NACK · Dữ liệu bị sửa (chữ ký vẫn đúng nhưng nội dung đã bị can thiệp) — xem log bên dưới.
